# HPC-Ready Biosignal AI Poster

This repository publishes the IHPCSS 5-page poster **HPC-Ready Biosignal AI** as a simple GitHub Pages site.

The poster presents an HPC-first framing of compute-efficient representation learning for long-sequence human biosignal AI, using ECG as the main experimental testbed. It highlights segment-aware quantisation, compact transformer modelling, calibration, transfer validation, and the path toward GPU profiling and scalable external validation.

## Updating The Poster

To publish a revised PDF, replace `hpc-ready-biosignal-ai-poster.pdf`, commit the change, and push to `main`. GitHub Pages automatically rebuilds the public viewer from the latest commit.
